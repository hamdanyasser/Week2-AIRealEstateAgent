# AI Real Estate Agent

Week 2 project for the SE Factory AIE Bootcamp.

A two-stage LLM + ML chain that predicts house prices from a natural
language description:

1. **Stage 1 (LLM):** extract structured features from a free-text query.
2. **ML model:** predict the price from the extracted features.
3. **Stage 2 (LLM):** interpret the prediction in plain English.

Detailed setup, architecture, and run instructions will be added in
Step 14.
